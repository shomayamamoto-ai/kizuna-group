# assets

このフォルダに画像を置くと、サイトへ自動反映されます（コード変更不要）。
いずれも未配置の場合は、上品な「絆」ゴールドのフォールバック表示になります。

| ファイル名 | 用途 | 推奨 |
| --- | --- | --- |
| `logo.png` | ヘッダー / ヒーロー / フッター / favicon | 背景黒 or 透過PNG。正方形に近いと綺麗 |
| `representative.jpg` | 代表紹介の顔写真 | 正方形・人物中央。円形に切り抜き表示されます |
| `office.jpg` | 会社概要のイメージ写真 | 横長推奨。暗めの写真がトーンに馴染みます |

## 反映方法（GitHub の Web からアップロードする場合）

1. リポジトリの `assets/` フォルダを開く
2. 右上「Add file」→「Upload files」
3. 上表のファイル名で画像をドラッグ＆ドロップ
4. `main` ブランチへ Commit

数十秒〜1分ほどで本番サイトに反映されます。
