# assets

このフォルダに画像を置くと、サイトへ**自動反映**されます（コード変更不要）。
未配置の場合は、上品な金色の「絆」エンブレム（SVG）が代わりに表示されます。

| ファイル名 | 用途 | 推奨スペック |
| --- | --- | --- |
| `logo.png` | ヘッダー / ヒーロー / フッター | 透過PNG（または背景黒）。正方形〜横長どちらも可。長辺 500〜1200px 程度、容量 1MB 以内が目安 |

## ロゴの反映方法（GitHub の Web からアップロード）

1. リポジトリの **`assets/`** フォルダを開く
2. 右上「**Add file**」→「**Upload files**」
3. ロゴ画像を **`logo.png`** という名前にして、ドラッグ＆ドロップ
4. ブランチ **`claude/laughing-cannon-dl85ho`**（または `main`）へ **Commit**

数十秒〜1分ほどで本番サイトに反映されます。
表示が変わらないときは、**スーパーリロード**（強制再読み込み）でご確認ください。

## 表示される場所と見え方

- **ヘッダー左上**：高さ 50px
- **ヒーロー中央**：幅 150〜230px の大きめ表示（ドロップシャドウ付き）
- **フッター**：高さ 84px

いずれも `logo.png` があればそれを優先表示し、無ければ金色エンブレムにフォールバックします。

## ファビコン（ブラウザのタブ用アイコン）

タブ用アイコンは `favicon.svg`（金色エンブレム）を使用しています。
ロゴ画像をタブにも使いたい場合はお知らせください。`favicon.png` を追加設定します。
