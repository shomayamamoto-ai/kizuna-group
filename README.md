# 株式会社SLC コーポレートサイト

「挑戦を繋ぎ、未来を創る」をコンセプトとした、株式会社SLCのコーポレートサイトです。

## デザイン

- **基調カラー**: ゴールド × ブラック（金黒）
- レスポンシブ対応（PC / タブレット / スマートフォン）
- スクロール連動アニメーション

## 構成

| ファイル | 内容 |
| --- | --- |
| `index.html` | ページ本体（マークアップ） |
| `styles.css` | スタイル（金黒テーマ） |
| `script.js` | ヘッダー追従・モバイルメニュー・スクロールアニメーション |
