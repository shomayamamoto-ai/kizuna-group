# 株式会社SLC コーポレートサイト

「挑戦を繋ぎ、未来を創る」をコンセプトとした、株式会社SLCのコーポレートサイトです。

🌐 公開URL: https://shomayamamoto-ai.github.io/kizuna-group/

## デザイン

- **基調カラー**: ゴールド × ブラック（金黒）
- プレミアムで上質なダークテーマ、和の高級感
- スクロール連動アニメーション／数値カウントアップ／ヒーローの光彩・パーティクル演出
- ローディング演出・スクロール進捗バー・トップへ戻るボタン
- レスポンシブ対応（PC / タブレット / スマートフォン）
- アクセシビリティ配慮（`prefers-reduced-motion` 対応）

## ロゴについて

ロゴ画像は `assets/logo.png` に配置すると、ヘッダー・ヒーロー・代表紹介・フッター・faviconへ自動で反映されます。
未配置の間は「絆」の金色マークに自動フォールバックします。詳細は `assets/README.md` を参照してください。

## 構成

| ファイル | 内容 |
| --- | --- |
| `index.html` | ページ本体（マークアップ） |
| `styles.css` | スタイル（金黒テーマ） |
| `script.js` | 各種インタラクション（ナビ・リビール・カウンター・パーティクル等） |
| `assets/` | ロゴなどの画像 |
| `.github/workflows/deploy-pages.yml` | GitHub Pages 自動デプロイ |

## セクション構成

1. ヒーロー（ロゴ／キャッチコピー／CTA）
2. 会社概要・哲学（実績数値カウンター付き）
3. 代表紹介（松下雄飛・沿革タイムライン）
4. 理念（ビジョン／目標）
5. 事業内容（6事業・絆BIZ LINKを強調）
6. お問い合わせ（CTA）

## デプロイ

`main` ブランチへ push すると GitHub Actions により自動でデプロイされます。

## ローカル表示

`index.html` をブラウザで開くだけで表示できます。
